>EN(translated) 
> What is Kazix?
This is [website](https://kahoot-god.github.io/kazix/),
which has two websites added:

>kahoot.it
>kahoot.rocks

> [Kahoot.it](https://kahoot.it):
can be used as a prelude to a real cheat (kahoot.rocks),
or you can play without cheats.
